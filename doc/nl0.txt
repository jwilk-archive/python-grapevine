=======
``nl0``
=======

return number items of an iterable (start with 0)

Usage
-----

* ``<iterable> | nl0``

Examples
--------

>>> from grapevine import *
>>> ('foo', 'bar', 'quux') | nl0 | list
[(0, 'foo'), (1, 'bar'), (2, 'quux')]
