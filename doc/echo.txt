========
``echo``
========

yield a single item

Usage
-----

* ``echo(<object>)``

Examples
--------

>>> from grapevine import *
>>> echo('quux') | list
['quux']

 * `GNU coreutils: echo <http://www.gnu.org/software/coreutils/manual/html_node/echo-invocation.html>`_
