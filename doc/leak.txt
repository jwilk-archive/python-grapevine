========
``leak``
========

remove magic

Usage
-----

* ``<pipelined-iterable> | leak``
* ``leak(<pipelined-iterable>)``

Examples
--------

>>> from grapevine import *

>>> leak(dev_null)  # doctest: +ELLIPSIS
<tupleiterator object at 0x...>

>>> cat([1, 2, 3]) | leak  # doctest: +ELLIPSIS
<listiterator object at 0x...>
