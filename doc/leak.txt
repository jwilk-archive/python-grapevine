========
``leak``
========

remove magic

Usage
-----

* ``<pipelined-iterable> | leak``
* ``leak(<pipelined-iterable>)``

Examples
--------

>>> from grapevine import *

>>> leak(dev_null)
<tuple_iterator object at 0xfeedface>

>>> cat([1, 2, 3]) | leak
<list_iterator object at 0xdeadf00d>
