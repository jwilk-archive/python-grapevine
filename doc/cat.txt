``cat``
=======

construct a pipeline

Usage
-----

* ``<iterable> | cat``
* ``cat(<iterable>â€¦)``

Examples
--------

>>> from grapevine import *
>>> [1, 2, 3] | cat | tuple
(1, 2, 3)
>>> cat([1, 2, 3]) | tuple
(1, 2, 3)
>>> cat([1, 2, 3], (4, 5, 6)) | tuple
(1, 2, 3, 4, 5, 6)
